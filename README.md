# AdvanDEB Architecture

This repository contains system-level design, planning, and documentation for the AdvanDEB project.

It covers:
- `advandeb-modeling-assistant`: Main platform GUI (frontend + backend) - user interface, chat, knowledge exploration, role-based access control.
- `advandeb-knowledge-builder`: Toolkit/package for knowledge operations - fact extraction, stylized facts, document ingestion, graph building.
- `advandeb-MCP`: Model Context Protocol server - exposes KB and MA tools via MCP for LLM agent workflows.

Use this repo as the single source of truth for:
- Overall system architecture and diagrams
- Cross-repository roadmaps and milestones
- Design decisions and integration contracts
- Development log and planning documents
